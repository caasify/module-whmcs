# Frontend Notes

This folder contains the React source for the CloudHub client dashboard.

## What Lives Here
- `src/` contains pages, components, context hooks, config, and styling.
- `public/` contains static frontend assets used during local development.
- `dist/` is not committed here. The production build is generated into the WHMCS package under `/Users/arzpmv/Desktop/projects/caasify-whmcs-module/build/whmcs/modules/addons/caasify/dist`.

## Local Work
- Run `npm install` inside this folder if dependencies are missing.
- Run `npm run dev` for local frontend development.
- Run `npm run lint` before packaging changes.

## Packaging
- Do not copy this source folder into WHMCS.
- Build output for WHMCS is generated by the root packaging flow, not by manually moving frontend files.
